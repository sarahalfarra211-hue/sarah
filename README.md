# منصة إعادة إعمار المنازل

## Problem (المشكلة)
تعاني الكثير من الأسر المتضررة من صعوبة فهم مراحل إعادة إعمار منازلهم،
وعدم وضوح التكاليف التقريبية، إضافة إلى عدم معرفة الجهات أو المهندسين
الذين يمكن التواصل معهم.

---

## Solution (الحل)
تقديم منصة ويب إرشادية تساعد المستخدم على:
- فهم مراحل إعادة الإعمار خطوة بخطوة
- معرفة التكاليف التقريبية حسب نوع الضرر
- الاطلاع على معلومات عن مهندسين مختصين
- التواصل بسهولة من خلال نموذج اتصال

---

## Target Users (المستخدم المستهدف)
- المتضررون من هدم أو تضرر المنازل
- أصحاب المنازل الراغبون في إعادة الإعمار
- أي شخص يبحث عن إرشادات أولية حول البناء

---

## Main Features (أهم الميزات)
- واجهة عربية بالكامل (RTL)
- عرض تفاعلي لمراحل إعادة الإعمار
- جدول يوضح التكاليف التقديرية
- قسم خاص بالمهندسين
- نموذج تواصل مع المستخدمين
- تصميم متجاوب باستخدام Bootstrap

---

## Technologies Used (التقنيات المستخدمة)
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery
- Font Awesome
- CDN Libraries

---

## How to Run the Project (طريقة التشغيل)
1. تحميل ملفات المشروع
2. فتح ملف `index.html`
3. تشغيله مباشرة في أي متصفح ويب
(Open index.html)
